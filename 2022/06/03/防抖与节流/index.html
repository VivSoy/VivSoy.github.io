防抖与节流的实现

通常项目实际开发过程中需要使用到防抖与节流的功能来进行优化，为了更好的理解防抖+节流的，通过两个及其通俗的例子来加深自己的印象

函数防抖

在联想搜索、请求登录、点击按钮的时候，为防止搜索项不停抖动和客户端重复发送请求，就会用到防抖。防抖是指的定义事件在设定的时间后执行回调，当设定时间未到时，该事件又再次被触发，就将定时器清空重新计时。

设置一个 input 控件，并对该控件的 input 事件进行监听。

通过控制台可以发现每输入一个字符，就会触发 input 的监听事件，打印输入框的内容。

函数节流

节流是指在规定的一个时间内，只能触发一次事件的回调函数，用户解决高频事件频繁触发事件而浪费的性能。
